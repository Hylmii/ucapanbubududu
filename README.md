# 한국어 과정 수료 축하 웹사이트

한국어 과정과 시험을 성공적으로 마친 분들을 위한 축하 웹사이트입니다.

## 프로젝트 설명

이 웹사이트는 한국어 학습 과정을 완료한 분들에게 축하 메시지를 전달하기 위해 제작되었습니다. 깔끔하고 우아한 디자인으로 진심 어린 축하의 마음을 담았습니다.

## 주요 특징

- 완전한 한국어로 작성
- 반응형 디자인
- 깔끔하고 미니멀한 레이아웃
- 보라색 그라데이션 배경
- 모바일 친화적

## 파일 구조

```
.
├── index.html          # 메인 HTML 파일
├── style.css           # 스타일시트
└── README.md           # 프로젝트 문서
```

## 사용 방법

1. 브라우저에서 `index.html` 파일을 열어주세요
2. 또는 Live Server를 사용하여 실행할 수 있습니다

## 기술 스택

- HTML5
- CSS3
- 반응형 웹 디자인

## 디자인 요소

- 그라데이션 배경 (#667eea → #764ba2)
- 한글 최적화 폰트 (맑은 고딕, Apple SD Gothic Neo)
- 카드 형태의 중앙 정렬 레이아웃
- 부드러운 그림자 효과

## 라이선스

이 프로젝트는 자유롭게 사용 가능합니다.
